# 🧠 FITCORE.AI

### Smart Fitness & Wellness Assistant

FITCORE.AI is a chatbot-based fitness and wellness assistant designed to help users maintain a healthy lifestyle by providing guidance on workouts, diet, and mental well-being in an interactive way.

---

## 🚀 Features

* 💬 Chat-based interaction
* 🏋️ Fitness and workout guidance
* 🥗 Basic diet and wellness suggestions
* ⚡ Fast and responsive system
* 🌐 Deployed on cloud

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **AI Processing:** Language model integration
* **Database:** SQLite
* **Deployment:** Render

---

## 📌 How It Works

1. User enters a query related to fitness or wellness
2. The system processes the request
3. Generates a structured response
4. Displays the response in chat format

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/fitcore-ai.git
cd fitcore-ai
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Set environment variables

```
GROQ_API_KEY=your_api_key_here
```

### 4. Run locally

```
python app.py
```
